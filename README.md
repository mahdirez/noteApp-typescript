# Sticky Notes App

A sticky notes app for adding, editing, deleting, and rearranging notes with drag-and-drop. Notes change color based on deadlines.

## Features

- Add notes
- Edit notes
- Delete notes
- Drag-and-drop to rearrange notes
- Visual indication of deadline status (expired or remaining days)

## Technologies Used

- React
- TypeScript
- Material UI
- @dnd-kit/core (for drag-and-drop functionality)

## Usage

- Adding Notes: Click the "Add Note" button, fill in the message and select the deadline, then submit the form.
- Editing Notes: Click the "Edit" button on a note, modify the message or deadline, and submit the form to save changes.
- Deleting Notes: Use the "Delete" button to remove a note.
- Rearranging Notes: Click and drag notes to change their order.
- Checking Status: Notes will display "Expired" if the deadline has passed. Otherwise, the remaining days are shown in yellow.
